# OpenRGB for Geek RGB 使用说明

## 安装
* 软件本身是绿色使用的，解压放在你常放应用程序的目录即可

* 首次启动需要，建议用管理员进行启动一次

* 如果启动遇到 dll 加载失败提示，通常是因为系统缺少 Microsoft Visual C++ 的运行库。可以到微软的官网下载 [vc-redist](
https://learn.microsoft.com/zh-cn/cpp/windows/latest-supported-vc-redist?view=msvc-170)
，安装后再启动 openRGB。
